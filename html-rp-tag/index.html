HTML-тег <rp>

HTML-тег <rp> используется для предоставления резервного текста для браузеров, не поддерживающих элемент <ruby> . Элемент <ruby> используется для отображения аннотаций, таких как руководства по произношению, для восточноазиатских языков(например, китайского, японского или корейского). Тег <rp> гарантирует, что содержимое останется читаемым в браузерах, которые не отображают элемент <ruby> должным образом.

Тег <rp> обычно содержит скобки или другие маркеры, которые заключают в себе текст Ruby, обеспечивая визуальный откат для неподдерживаемых сред.

Базовый синтаксис HTML-тега <rp>

Тег <rp> используется внутри элемента <ruby> вместе с тегом <rt> (текст Ruby), как показано: 

<ruby>
    Base Text
    <rp>(</rp>
    <rt>Ruby Text</rt>
    <rp>)</rp>
</ruby>

Тег <rp> заключает текст Ruby в скобки для браузеров, которые не поддерживают аннотации Ruby.

Пример использования тега <rp>

Вот пример того, как работает тег <rp> внутри элемента <ruby> : 

<!DOCTYPE html>
<html>
    <body>
        <h2>Ruby Annotation Example</h2>
        <p>
            <ruby>
                漢字
                <rp>(</rp>
                <rt>kanji</rt>
                <rp>)</rp>
            </ruby>
        </p>
    </body>
</html>

Пояснение: В браузерах, поддерживающих элемент <ruby> , «kanji» отображается как руководство по произношению над «漢字». В браузерах, не поддерживающих этот элемент, текст будет отображаться как «漢字(kanji)».

Атрибуты HTML-тега <rp>

  • Нет атрибутов: тег <rp> не имеет каких-либо определенных атрибутов.
  • Глобальные атрибуты: поддерживает все глобальные атрибуты, такие как id , class и style .

Практическое применение тега <rp>

  • Резервный текст: убедитесь, что аннотации Ruby остаются читаемыми в браузерах, которые не поддерживают элемент <ruby> .
  • Совместимость со всеми браузерами: обеспечивайте удобную альтернативу для старых браузеров, сохраняя при этом семантическую целостность.
  • Доступность: Улучшите читаемость для пользователей, которым для понимания аннотаций Ruby необходимы визуальные подсказки, такие как скобки.
