HTML-тег <cite>

HTML-тег <cite> используется для определения названия творческого произведения, такого как книга, статья, фильм, телешоу, песня и т. д. Это семантический тег, указывающий на цитату или ссылку на другой источник.

По умолчанию браузеры часто отображают содержимое тега <cite> курсивом, хотя это можно настроить с помощью CSS.

Базовый пример HTML-тега <cite>

Вот простой пример использования тега <cite> : 

<!DOCTYPE html>
<html>
    <body>
        <h2>Cite Example</h2>
        <p>One of my favorite books is <cite>To Kill a Mockingbird</cite> by Harper Lee.</p>
    </body>
</html>

Пояснение: В этом примере название книги заключено в тег <cite> , чтобы указать, что это творческая работа.

Стилизация тега <cite>

Стиль по умолчанию для тега <cite> — курсив. Вы можете настроить его внешний вид с помощью CSS: 

<!DOCTYPE html>
<html>
    <head>
        <style>
            cite {
                font-style: normal;
                font-weight: bold;
                color: #007BFF;
            }
        </style>
    </head>
    <body>
        <h2>Cite Example</h2>
        <p>One of my favorite books is <cite>To Kill a Mockingbird</cite> by Harper Lee.</p>
    </body>
</html>

Результат: заголовок внутри тега <cite> отображается с пользовательским стилем, примененным через CSS.

Особые случаи для HTML-тега <cite>

Вложенное использование тега цитирования

Вы можете вложить тег <cite> в другие элементы, например в блок цитат, для указания источника цитаты: 

<!DOCTYPE html>
<html>
    <body>
        <h2>Cite in Blockquote</h2>
        <blockquote>
            "In the middle of every difficulty lies opportunity." 
            <cite>Albert Einstein</cite>
        </blockquote>
    </body>
</html>

Пояснение: Тег <cite> здесь указывает автора цитаты внутри элемента blockquote.

Ссылаясь на веб-сайты с использованием тега cite

Вы можете использовать тег <cite> для ссылки на веб-сайты или онлайн-статьи: 

<!DOCTYPE html>
<html>
    <body>
        <h2>Citing a Website</h2>
        <p>For more information, visit <cite><a href="https://www.wikipedia.org">Wikipedia</a></cite>.</p>
    </body>
</html>

Пояснение: В этом примере тег <cite> используется с гиперссылкой для цитирования определенного веб-сайта.

Советы по доступности тега <cite>

При использовании тега <cite> :

  • Убедитесь, что цитируемая работа правильно идентифицирована(например, по названию или URL).
  • Объедините тег <cite> с семантическими элементами, такими как <blockquote> или <a> для лучшего контекста.
