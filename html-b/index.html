HTML-элемент полужирного шрифта

Тег HTML Bold <b> используется для выделения текста жирным шрифтом в HTML-документе. Обратите внимание, что использование тега Bold для выделения некоторых фрагментов текста не делает этот текст более важным.

Пример

В следующем примере показан простой элемент Bold. 

<!DOCTYPE html>
<html>
    <body>
        <p>normal text <b>bold text</b></p>
    </body>
</html>

Примечание: элемент HTML Bold начинается с тега <b> и заканчивается конечным тегом </b> .

CSS по умолчанию для полужирного шрифта

По умолчанию для элемента Bold установлены следующие свойства CSS.

Копировать 
font-weight: bold;

Встроенный стиль для HTML-жирного шрифта

Мы можем изменить стиль элемента Bold с помощью встроенного стиля, используя атрибут style.

В следующем примере мы установили красный цвет для элемента Bold. 

<!DOCTYPE html>
<html>
    <body>
        <p>normal text <b style="color:red;">bold text</b></p>
    </body>
</html>

Применить CSS для жирного элемента

Мы можем применить CSS ко всем элементам Bold, используя тег b . 

<!DOCTYPE html>
<html>
    <head>
        <style>
            b {
              color:red;
            }
          </style>
    </head>
    <body>
        <p>normal text <b>bold text</b></p>
    </body>
</html>
