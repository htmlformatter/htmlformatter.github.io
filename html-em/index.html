HTML-элемент выделенного текста

Тег HTML Emphasised <em> используется для определения выделенного текста в HTML-документе.

Содержание
  • Пример
  • CSS по умолчанию для HTML-элемента <em>
  • Встроенный стиль для выделенного элемента HTML
  • Применить CSS для выделенного элемента HTML
  • Заключение

Пример

Простой выделенный элемент показан в следующем примере. 

<!DOCTYPE html>
<html>
    <body>
        <p>normal text <em>emphasized text</em></p>
    </body>
</html>

Примечание: Выделенный элемент HTML начинается с тега <em> и заканчивается конечным тегом </em> .

CSS по умолчанию для HTML-элемента <em>

По умолчанию для выделенного элемента установлены следующие свойства CSS.

Копировать 
font-style: italic;

Встроенный стиль для выделенного элемента HTML

Мы можем изменить стиль выделенного элемента с помощью встроенного стиля, используя атрибут style.

В следующем примере мы установили красный цвет для выделенного элемента. 

<!DOCTYPE html>
<html>
    <body>
        <p>normal text <em style="color:red;">emphasized text</em></p>
    </body>
</html>

Применить CSS для выделенного элемента HTML

Мы можем применить CSS для всех выделенных элементов, используя тег em . 

<!DOCTYPE html>
<html>
    <head>
        <style>
            em {
              color:red;
            }
          </style>
    </head>
    <body>
        <p>normal text <em>emphasized text</em></p>
    </body>
</html>
