HTML-элемент заголовка 3

Тег HTML-заголовка 3 <h3> используется для определения заголовка типа 3 в HTML-документе.

Содержание
  • Пример
  • CSS по умолчанию для HTML <h3>
  • Встроенный стиль для элемента HTML-заголовка 3

  • Заключение

Пример

Простой элемент «Заголовок 3» показан в следующем примере. 

<!DOCTYPE html>
<html>
    <body>
        <h3>Hello World</h3>
    </body>
</html>

Примечание: элемент HTML Heading 3 начинается с тега <h3> и заканчивается конечным тегом </h3> .

CSS по умолчанию для HTML <h3>

По умолчанию для элемента Heading 3 установлены следующие свойства CSS.

Копировать 
display: block;
font-size: 1.17em;
margin-block-start: 1em;
margin-block-end: 1em;
margin-inline-start: 0px;
margin-inline-end: 0px;
font-weight: bold;

Встроенный стиль для элемента HTML-заголовка 3

Мы можем изменить стиль элемента «Заголовок 3» с помощью встроенного стиля, используя атрибут style.

В следующем примере мы установили красный цвет для элемента Heading 3. 

<!DOCTYPE html>
<html>
    <body>
        <h3 style="color:red;">Hello World</h3>
    </body>
</html>

Применить CSS к элементу HTML-заголовка 3

Мы можем применить CSS ко всем элементам заголовка 3, используя тег h3 . 

<!DOCTYPE html>
<html>
    <head>
        <style>
            h3 {
              color:red;
            }
          </style>
    </head>
    <body>
        <h3>Hello World</h3>
    </body>
</html>
