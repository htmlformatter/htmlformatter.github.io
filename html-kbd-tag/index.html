HTML-тег <kbd>

HTML-тег <kbd> используется для определения ввода с клавиатуры. Он представляет текст, который пользователь должен ввести с помощью клавиатуры, голосового ввода или любого другого устройства ввода текста. По умолчанию браузеры обычно отображают текст внутри тега <kbd> моноширинным шрифтом, что визуально отличает его от остального содержимого.

Тег <kbd> часто используется в учебных пособиях, документации или интерактивных приложениях для обозначения команд, сочетаний клавиш или входных данных.

Базовый синтаксис тега HTML <kbd>

Базовая структура тега <kbd> : 

<kbd>Keyboard input</kbd>

Текст внутри тега <kbd> будет отображаться моноширинным шрифтом, чтобы указать, что он представляет собой пользовательский ввод.

Базовый пример HTML-тега <kbd>

Вот простой пример использования тега <kbd> для указания сочетания клавиш: 

<!DOCTYPE html>
<html>
    <body>
        <p>To save a file, press <kbd>Ctrl</kbd> + <kbd>S</kbd>.</p>
    </body>
</html>

Пояснение: Тег <kbd> используется для представления клавиш клавиатуры «Ctrl» и «S» в качестве пользовательского ввода для операции сохранения.

Использование тега <kbd> в формах или приложениях

Тег <kbd> полезен для предоставления инструкций пользователям. Например: 

<!DOCTYPE html>
<html>
    <body>
        <h2>Shortcut Instructions</h2>
        <p>Use the following shortcuts for navigation:</p>
        <ul>
            <li>To open the help menu, press <kbd>F1</kbd>.</li>
            <li>To refresh the page, press <kbd>Ctrl</kbd> + <kbd>R</kbd>.</li>
            <li>To search, press <kbd>Ctrl</kbd> + <kbd>F</kbd>.</li>
        </ul>
    </body>
</html>

Пояснение: Тег <kbd> используется в списке для представления сочетаний клавиш в структурированном и визуально понятном виде.

Стилизация тега <kbd> с помощью CSS

Моноширинный шрифт по умолчанию для тега <kbd> можно настроить с помощью CSS. Вот пример: 

<!DOCTYPE html>
<html>
    <head>
        <style>
            kbd {
                background-color: #f4f4f4;
                border: 1px solid #ccc;
                border-radius: 4px;
                padding: 2px 4px;
                font-family: 'Courier New', monospace;
                color: #333;
            }
        </style>
    </head>
    <body>
        <p>To undo an action, press <kbd>Ctrl</kbd> + <kbd>Z</kbd>.</p>
    </body>
</html>

Результат: ввод с клавиатуры отображается в стилизованном виде, включая светлый фон, границу и отступы, имитирующие клавиши физической клавиатуры.

Практическое применение тега <kbd>

  • Руководства пользователя: Укажите сочетания клавиш и команды в документации по программному обеспечению.
  • Интерактивные уроки: научите пользователей выполнять задачи с помощью определенных комбинаций клавиш.
  • Доступность: предоставьте понятные инструкции для пользователей, использующих клавиатуру для навигации.
  • Руководства по играм: список клавиатурных элементов управления для игр или интерактивных приложений.
