HTML <strong>

Тег HTML Strong <strong> используется для определения важного текста в HTML-документе.

Содержание
  • Пример
  • CSS по умолчанию для HTML <strong>
  • Встроенный стиль для HTML Strong Element
  • Применить CSS для HTML Strong Element
  • Заключение

Пример

В следующем примере показан простой элемент Strong. 

<!DOCTYPE html>
<html>
    <body>
        <p>normal text <strong>important text</strong></p>
    </body>
</html>

Примечание: элемент HTML Strong начинается с тега <strong> и заканчивается конечным тегом </strong> .

CSS по умолчанию для HTML <strong>

По умолчанию для элемента Strong установлены следующие свойства CSS.

Копировать 
font-weight: bold;

Встроенный стиль для HTML Strong Element

Мы можем изменить стиль элемента Strong с помощью встроенного стиля, используя атрибут style .

В следующем примере мы установили красный цвет для элемента Strong. 

<!DOCTYPE html>
<html>
    <body>
        <p>normal text <strong style="color:red;">important text</strong></p>
    </body>
</html>

Применить CSS для HTML Strong Element

Мы можем применить CSS ко всем элементам Strong, используя имя тега strong . 

<!DOCTYPE html>
<html>
    <head>
        <style>
            strong {
              color:red;
            }
          </style>
    </head>
    <body>
        <p>normal text <strong>important text</strong></p>
    </body>
</html>
