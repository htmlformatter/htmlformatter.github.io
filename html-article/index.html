HTML <статья>

Тег HTML-статьи <article> используется для определения независимого, самодостаточного содержимого в HTML-документе.

Тег статьи обычно используется для сообщений на форумах, сообщений в блогах, новостных сюжетов и т. д.

Содержание
  • Пример
  • CSS по умолчанию для HTML <article>
  • Встроенный стиль для элемента HTML-статьи

  • Заключение

Пример

В следующем примере показан простой элемент Article. 

<!DOCTYPE html>
<html>
    <body>
        <article>
            <h2>Topic 1</h2>
            <p>This is a sample text.</p>
        </article>
    </body>
</html>

Примечание: HTML-элемент статьи начинается с тега <article> и заканчивается конечным тегом </article> .

В одном документе может быть одна или несколько статей, как показано ниже. 

<!DOCTYPE html>
<html>
    <body>
        <article>
            <h2>Topic 1</h2>
            <p>This is a sample text.</p>
        </article>
        <article>
            <h2>Topic 2</h2>
            <p>This is some sample text.</p>
        </article>
    </body>
</html>

CSS по умолчанию для HTML <article>

По умолчанию для элемента Article установлены следующие свойства CSS.

Копировать 
display: block;

Встроенный стиль для элемента HTML-статьи

Мы можем изменить стиль элемента «Статья» с помощью встроенного стиля, используя атрибут style .

В следующем примере мы установили красный цвет для элемента Article. 

<!DOCTYPE html>
<html>
    <body>
        <article style="color:red">
            <h2>Topic 1</h2>
            <p>This is a sample text.</p>
        </article>
    </body>
</html>

Применить CSS к элементу HTML-статьи

Мы можем применить CSS ко всем элементам статьи, используя имя тега article . 

<!DOCTYPE html>
<html>
    <head>
        <style>
            article {
              color:red;
            }
          </style>
    </head>
    <body>
        <article>
            <h2>Topic 1</h2>
            <p>This is a sample text.</p>
        </article>
        <article>
            <h2>Topic 2</h2>
            <p>This is some sample text.</p>
        </article>
    </body>
</html>
